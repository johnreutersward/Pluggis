Pluggis
=======

Plugin-based IRC-BOT built with .NET

Features
--------
* URL shortener
* Ragefac.es identifier
* JSON settings file 

Dependencies
------------
* [SmartIrc4net](http://www.meebey.net/projects/smartirc4net/)
* [JSON.NET](http://json.codeplex.com/)
* [log4net](http://logging.apache.org/log4net/)
